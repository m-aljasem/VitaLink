# Image Placeholders

This directory contains placeholder images for the VitaLink website. Replace these with actual screenshots when available.

## Required Images

### App Screenshots
- `placeholder-hero.png` - Hero section app screenshot (recommended: 300x600px or similar phone aspect ratio)
- `placeholder-dashboard.png` - Home dashboard screenshot (recommended: 400x600px)
- `placeholder-analytics.png` - Analytics/metrics detail screenshot (recommended: 400x600px)
- `placeholder-provider.png` - Provider dashboard screenshot (recommended: 400x600px)
- `placeholder-language.png` - Language selection screenshot (recommended: 400x600px)
- `placeholder-entry.png` - Data entry form screenshot (recommended: 400x600px)
- `placeholder-reminders.png` - Reminders screen screenshot (recommended: 400x600px)

### Social Media Preview Images
- `og-image.png` - Open Graph image for Facebook/LinkedIn sharing (recommended: 1200x630px)
- `twitter-image.png` - Twitter Card image (recommended: 1200x630px)

### Logo
- `vitalink.png` - Main app logo (already included)

## Image Guidelines

- Format: PNG or JPG
- Aspect ratio: Maintain phone/tablet aspect ratios for screenshots
- File size: Optimize images for web (aim for < 500KB each)
- Quality: Use high-quality screenshots that showcase the app's design
- Social images: Should include VitaLink branding and key messaging

The website will automatically fall back to Font Awesome icons if images are not found.
